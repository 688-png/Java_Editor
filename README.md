# JavaLab - Mobile Responsive IDE

## Setup
1. `npm install`
2. Open two terminals:
   - Terminal 1: `npm run dev` (Frontend)
   - Terminal 2: `node server.js` (Backend)

## Features
- Full-screen code editor with Monaco.
- **Mobile Optimized**: Tabbed switching between Editor and Terminal.
- **Import/Export**: Load local `.java` files or download your progress.
- Local Storage persistent sessions.
- Dark theme inspired by VS Code/Replit.
